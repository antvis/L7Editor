## 📦 组件

```bash
npm install @antv/l7-editor -S
```

```jsx | pure
import { L7Editor } from '@antv/l7-editor';

export default () => {
  return <L7Editor autoFitBounds activeTab="geojson" theme="dark" />;
};
```

## ⌨️ 本地开发

```bash
# 安装依赖
npm install

# 工具页面运行页面
npm run start

# 工具页面打包
npm run build
```
